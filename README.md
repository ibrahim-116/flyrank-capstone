E-commerce Website 

-A react, react-router website to sell products listed with a separate admin panel

Tech Stack:
-React: Major frontend framework
-Tailwind: CSS helper framework
-ShadCN: Library for reusable components

Getting Started:
-clone this repository using: git clone <github repo URL>
-Setup you .env file
-Run the command "npm run dev" or "npm run build && npm run start" to run this project yourself
